Установка виртуальной среды:

Для Винды из Powershell:

pip install virtualenv - если не стои пакет

В папке проекта:

python -m venv "Имя среды"

Результат - появление в папке проекта папки с анально огороженной версией Питона

cd "Имя среды"	|
cd scripts	| - запустить activate.bat - результат, изменение перменных среды
activate	|

pip install django (любые нужные пакеты)
все пакеты должны устанавливаться в папку "Имя среды", если нет - менять переменные вручную (VIRTUALENV, PATH).


Создание проекта:

В папке проекта:
django-admin startproject "Имя проекта"

Запуск сервера:
python manage.py runserver [порт]


Для Гита:

Поставить Git Bash

После установки пакетов:
pip freeze > requirements.txt
Создает файл со списком необходимых пакетов

При переносе в другую виртуальную среду:

pip install -r requirements.txt -будут установлены все необходимые пакеты

в папке "имя проекта" создать .gitignore - файл со списком шаблонов игнорируемых гитом файлов

в папке "имя проекта"Ж

git init - создает локальный репозиторий в папке

git add --all добавляет все файлы, кроме тех, шаблоны которых перечислены в .gitignore в список отслеживаемых

git remote add origin <ссылка> - связывает локальный репозиторий с удаленным

git commit -m "комментарий" - коментарий к изменениям

git push origin master - отправка на удаленный репозиторий
